H3C NX30Pro еҲ· openwrt ж•ҷзЁӢ

еҸ‘иЎЁдәҺ2023-07-10|жӣҙж–°дәҺ2024-03-28|и·Ҝз”ұеҷЁ
|еӯ—ж•°жҖ»и®Ў:938|йҳ…иҜ»ж—¶й•ҝ:3еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

H3C NX30 Pro е°ұжҳҜзӣ®еүҚзҷҫе…ғи·Ҝз”ұеҷЁдёӯжҖ§д»·жҜ”жңҖй«ҳзҡ„пјҲйҷӨдәҶиҝҗиҗҘе•Ҷе®ҡеҲ¶жңәпјүпјҢйқһеёёжҺЁиҚҗгҖӮH3C NX30 Pro е’Ң 360 T7гҖҒе°Ҹзұі WR30U зҡ„зЎ¬д»¶й…ҚзҪ®е®Ңе…ЁдёҖж ·пјҢз”Ёзҡ„йғҪжҳҜзҷҫе…ғи·Ҝз”ұеҷЁзҘһU MT7981BпјҢarm A53 еҸҢж ё 1.3GhzпјҢ256M зҡ„еҶ…еӯҳпјҢд№ҹйғҪжҳҜеҶ…зҪ®еҠҹж”ҫ Wi-Fi дҝЎеҸ·жІЎе·®пјҢйғҪжҳҜ 200 е…ғеҶ…жңҖеҘҪгҖӮH3C NX30 Pro дё»иҰҒдјҳеҠҝжҳҜйқһиҝҗиҗҘе•Ҷе®ҡеҲ¶йҷҗеҲ¶е°‘пјҢиҙӯд№°е’Ңдҝқдҝ®йғҪж–№дҫҝпјҢзӣ®еүҚеҲ·жңәд№ҹжҳҜжңҖж–№дҫҝзҡ„гҖӮ

OpenWrt

еӣәд»¶йғЁеҲҶжҲӘеӣҫ


еҮҶеӨҮе·ҘдҪң

  • H3C NX30 Pro и·Ҝз”ұеҷЁ
  • жңүзҪ‘еҸЈзҡ„ Windows з”өи„‘пјҲдё»иҰҒз”Ё telnetгҖҒsshпјҢеҲ«зҡ„зі»з»ҹд№ҹиЎҢпјү
  • дёӢиҪҪйңҖиҰҒзҡ„ж–Үд»¶ пјҲеҢ…еҗ«дәҶ putty.exeгҖҒwinscpгҖҒе’Ңеӣәд»¶пјү
  • еҰӮең°еқҖеӨұж•ҲпјҢеҠ зҫӨиҺ·еҸ–еҚіеҸҜгҖӮ

ејҖеҗҜ SSH

иҝҷдёҖжӯҘиҜ·зЎ®дҝқи·Ҝз”ұеҷЁиғҪжӯЈеёёиҒ”зҪ‘гҖӮNX30 Pro й»ҳи®ӨејҖеҗҜдәҶ telnetпјҢй»ҳи®Өзҡ„ең°еқҖжҳҜ 192.168.124.1пјҢз”ЁжҲ·еҗҚжҳҜ H3CпјҲе…ЁеӨ§еҶҷпјүпјҢеҜҶз Ғе°ұдҪ и®ҫзҪ®зҡ„и·Ҝз”ұеҷЁеҗҺеҸ°еҜҶз ҒпјҢз«ҜеҸЈжҳҜ 99гҖӮ

жү“ејҖ termius йҖүжӢ© New Host ж·»еҠ пјҢAddress еЎ«еҶҷ 192.168.124.1пјҢеҸ–ж¶ҲеӢҫйҖү SSHпјҢеӢҫйҖү TelnetпјҢPort з«ҜеҸЈеЎ«еҶҷдёҠ 99гҖӮ


з„¶еҗҺиҫ“е…Ҙз”ЁжҲ·еҗҚе’ҢеҜҶз ҒпјҢиҫ“е…ҘеҜҶз ҒдёҚдјҡжҳҫзӨәпјҢеӣһиҪҰеҚіеҸҜиҝһдёҠ TelnetгҖӮ

з„¶еҗҺеӨҚеҲ¶дёӢйқўзҡ„д»Јз ҒеҲ°з»Ҳз«ҜйҮҢжү§иЎҢеҚіеҸҜејҖеҗҜ SSHпјҡ

еӨҮд»Ҫзі»з»ҹ

еңЁз»Ҳз«ҜйҮҢжү§иЎҢиҝҷе‘Ҫд»ӨпјҢе°Ҷзі»з»ҹеӨҮд»ҪеҲ° tmp и·Ҝеҫ„дёӢпјҢеӨҮд»Ҫзҡ„иҝҮзЁӢжҜ”иҫғж…ўпјҢйңҖиҰҒ 1-2 еҲҶй’ҹгҖӮ

з„¶еҗҺжү“ејҖ WinSCPпјҢж–Үд»¶еҚҸи®®йҖүжӢ© SCPпјҢдё»жңәеҗҚ 192.168.124.1пјҢз«ҜеҸЈеҸ· 22пјҢз”ЁжҲ·еҗҚ H3CпјҢз„¶еҗҺзҷ»еҪ•иҫ“е…ҘеҜҶз Ғе°ұиғҪиҝһдёҠгҖӮ

зҷ»еҪ•еҗҺпјҢе°ҶеҸідҫ§ NX30Pro tmp и·Ҝеҫ„дёӢзҡ„ backup.img ж–Үд»¶жӢ–еҲ°з”өи„‘жЎҢйқўдҝқеӯҳпјҢеҲ·еӣһе®ҳж–№еӣәд»¶йңҖиҰҒз”ЁеҲ°гҖӮ

еҲ·еҶҷ uboot

дёӨз§Қж–№ејҸдәҢйҖүе…¶дёҖеҚіеҸҜпјҢжӣҙжҺЁиҚҗж–№жі•дёҖпјҢеӣ дёәжңүеҸҜиғҪ curl еӣ дёәзҪ‘з»ңй—®йўҳж— жі•жӯЈеёёдёӢиҪҪ ubootгҖӮ


е°ҶдёӢиҪҪеҘҪзҡ„ uboot.bin ж–Үд»¶д»Һе·Ұиҫ№з”өи„‘жӢ–иҝӣеҸідҫ§и·Ҝз”ұеҷЁ tmp и·Ҝеҫ„дёӢгҖӮ
з„¶еҗҺжү§иЎҢе‘Ҫд»ӨпјҢеҰӮеӣҫжүҖзӨәе®ҢжҲҗдәҶеҶҷе…Ҙ ubootгҖӮпјҲиҜ·зЎ®дҝқMD5 ж ЎйӘҢз»“жһңе’ҢеӣҫдёӯзӣёеҗҢпјҢеҶҚж•ІеӣһиҪҰжү§иЎҢпјҢ еҗҰеҲҷдјҡеҸҳз – )

еҲ·еҶҷ openwrt

и·Ҝз”ұеҷЁж–ӯз”өеҗҺпјҢе…ҲжҢүдҪҸиғҢеҗҺ Reset жҒўеӨҚжҢүй’®дёҚж”ҫпјҢеҶҚжҸ’з”өпјҢзӯүеҫ… 10s е·ҰеҸіжқҫејҖиғҢеҗҺ ResetпјҢи·Ҝз”ұеҷЁе°ұиҝӣе…ҘдәҶ ubootпјҢз”өи„‘з”ЁзҪ‘зәҝиҝһжҺҘи·Ҝз”ұеҷЁ LAN1пјҢе№¶и®ҫзҪ®еҘҪйқҷжҖҒ IPпјҡIPең°еқҖеЎ« 192.168.1.2пјҢеӯҗзҪ‘жҺ©з Ғ 255.255.255.0пјҢзҪ‘е…і 192.168.1.1пјҢDNS 192.168.1.1гҖӮ

зҺ°еңЁжөҸи§ҲеҷЁжү“ејҖ 192.168.1.1 е°ұиғҪжү“ејҖ uboot еҗҺеҸ°гҖӮ

йҖүжӢ©дёӢиҪҪеҘҪзҡ„ factory.binзҡ„еӣәд»¶пјҢupload дёҠдј еҗҺпјҢupdate жӣҙж–°пјҢеҶҷе…ҘеҚіеҸҜгҖӮ

жӣҙж–°е®ҢжҲҗеҗҺзі»з»ҹдјҡйҮҚеҗҜиҝӣе…Ҙ openwrt зҡ„зі»з»ҹпјҢй»ҳи®Өзҡ„ WiFi жҳҜ MTK ејҖеӨҙжІЎжңүеҜҶз ҒпјҢеҗҺеҸ°ең°еқҖ 192.168.5.1пјҢз”ЁжҲ·еҗҚ rootпјҢеҜҶз Ғ passwordгҖӮ

еҲ·еӣһе®ҳж–№

еҲ·еӣһе®ҳж–№йқһеёёз®ҖеҚ•пјҢиҝӣе…Ҙ uboot еҗҺйҖүжӢ©д№ӢеүҚеӨҮд»Ҫзҡ„ backup.img ж–Үд»¶жӣҙж–°еҚіеҸҜпјҢзі»з»ҹе°ұдјҡйҮҚеҗҜиҝӣе…Ҙе®ҳж–№зҡ„зі»з»ҹгҖӮ

жҖ»з»“

H3C NX30 Pro е’Ңд№ӢеүҚжҺЁиҚҗзҡ„дёӨж¬ҫ MT7981B зҡ„и·Ҝз”ұй…ҚзҪ®жІЎеҢәеҲ«пјҢйқһиҝҗиҗҘе•Ҷе®ҡеҲ¶пјҢе’ҢжңҖж–№дҫҝзҡ„еҲ·жңәе°ұжҳҜжңҖеӨ§зҡ„дјҳеҠҝпјҢдёӨзҷҫе…ғд»·ж јеҶ… MT7981B зҡ„и·Ҝз”ұеҷЁжңүзқҖжңҖдјҳз§Җзҡ„ WiFi дҝЎеҸ·пјҢиҝҳз®—дёҚй”ҷзҡ„ CPU жҖ§иғҪпјҢдҪҶжҲ‘и®ӨдёәжңҖйҮҚиҰҒзҡ„жҳҜиғҪеҲ·жңәе№¶дё”жңүеҘҪзҡ„зӨҫеҢәж”ҜжҢҒгҖӮ
